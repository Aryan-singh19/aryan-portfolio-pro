# Aryan Portfolio Pro

Premium one-page portfolio website for Aryan Singh.

## Live Demo

`https://aryan-singh19.github.io/aryan-portfolio-pro/`

## Run Locally

Open `index.html` in a browser.

## XAMPP Deploy Path

Copy project into `/opt/lampp/htdocs/aryan-portfolio-pro` and open:

`http://localhost/aryan-portfolio-pro`
